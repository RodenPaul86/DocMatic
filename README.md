## Features

- Ad-free.
- Regular, Strobing, and Flashing LED Light.
- Screen Light: The user uses the screen for light.
- On-board map with turn-by-turn directions.
- Morse Code.

## Requirements

- Xcode 14.3.1
- iOS 12+
